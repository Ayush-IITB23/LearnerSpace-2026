## 📝 Assignment

Build a **Login Page** using HTML & CSS.

### 🖼️ How Your Page Should Look

- Form should be centered on the page 
- Email and Password input fields
- A Login button
- "Forgot Password?" link

Submit your work in the following structure:

```

├── index.html

```
